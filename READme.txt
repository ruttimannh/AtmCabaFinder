Para ejecutar la API solo basta con ejecutar atmCabaFinder.py

La API se conecta a la base de datos cajeros.db para acceder a los datos de los cajeros. El .csv solo lo use para leerlo
y crear la base de datos. Puedo reescribir ese codigo y mandarlo si es necesario.

El archivo recargas.txt se utiliza solo para guardar la fecha de la ultima recarga, y en caso de que cumpla con los
requisitos, efectuarla en el codigo.
